Input File should have three sheets with names:
movements
outlets
products

movements sheet:
Sheet header values should have no space between and around. Select first row, press Ctrl+H and remove spaces.
Do same for outlets and products sheets.

In movements sheet:
After OutletName Column, RouteCode column is available. Use Vlookup formula to set RouteCode from outlets sheet.
Select the RouteCode column and press Ctrl+C for copy purpose and then press Ctrl+Alt+V. 
Select Value and then press Ok.

Now, your input file is ready!
Open the program, upload your input file and save output file in desired folder!
